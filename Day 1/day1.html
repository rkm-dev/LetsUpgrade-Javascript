Q2 :

Write the difference between var, let and const with code examples

Ans : var was introduced in js for variable declaration, which has a local and a global scope.

let is now preferred for variable declaration. It also solves the problem that comes with var. let is block scoped. let can be updated but not re-declared. let declarations are hoisted to the top. Unlike var which is initialized as undefined, the let keyword is not initialized. So if you try to use a let variable before declaration, you'll get a Reference Error.

Variables declared with the const maintain constant values. const declarations are block scoped. const cannot be updated or re-declared.
