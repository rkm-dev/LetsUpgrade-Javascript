Question 2

Explain with examples the remaining methods of String and Array

Ans : A string can be converted to an array with the split() method: "hello, hey, hi".split()

The charCodeAt() method returns the unicode of the character at a specified index in a string. The method returns a UTF-16 code (an integer between 0 and 65535).

The charAt() method returns the character at a specified index (position) in a string

The trim() method removes whitespace from both sides of a string

The concat() method can be used instead of the plus operator.

A string is converted to lower case with toLowerCase() and to uppercase with toUpperCase()

substring or substr()is similar to slice since it slices the string with the given start and stop end points

=================================================================================================

Array methods

The fill() method fills the specified elements in an array with a static value.

The filter() method creates an array filled with all array elements that pass a test.

The find() method returns the value of the first element in an array that pass a test.

The findIndex() method returns the index of the first element in an array that pass a test.
